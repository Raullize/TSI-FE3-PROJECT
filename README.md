# AutoHub

AutoHub é uma aplicação web que simula a vitrine de uma concessionária de veículos. Seus objetivos são permitir que os usuários explorem marcas e modelos de carros vendidos no Brasil, consultem os preços de referência pela Tabela FIPE e visualizem um estoque simulado de veículos.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

## Como Executar

1. Clone este repositório.
2. Abra o arquivo `index.html` em seu navegador.
